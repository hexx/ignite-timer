# プロジェクト概要
- アプリケーション名: ignite-timer
- 目的: ユーザーの行動を促すための3つの心理学的アプローチに基づいたタイマーアプリ

# 技術スタック
- インフラ / デプロイ: Cloudflare (Cloudflare Pages or Workers)
- バックエンド / API: Hono
- フロントエンド: React (Vite)
- ユニットテスト: Vitest
- E2Eテスト: Playwright
- 言語: TypeScript

# 機能要件（3つのタイマーモード）
以下の3つのモードを備えたUIを作成してください。各モードはシームレスに切り替えられるようにします。

1. 【5秒ルールモード】(The 5-Second Rule)
- 目的: オートパイロットの遮断し、即座に行動を開始させる。
- 動作: 「5, 4, 3, 2, 1」とカウントダウンするシンプルなタイマー。
- UI: カウントダウン中は視覚的に切迫感を与え、0になった瞬間に「GO!」などの強いメッセージを表示する。

2. 【2分間ルールモード】(The 2-Minute Rule)
- 目的: タスクの初期起動に必要な摩擦を下げ、行動を開始した事実を作る。
- 動作: 2分間（120秒）のタイマー。
- UI: 「とりあえず手を動かそう（ゲートウェイ・ハビット）」というメッセージとともに進行状況がわかるプログレスバーを表示する。

3. 【10分間ルールモード】(The 10-Minute Rule)
- 目的: ツァイガルニク効果と行動活性化を利用し、作業の継続を促す。
- 動作: 10分間（600秒）のタイマー。
- UI: 「10分だけ全力でやろう。その後はやめてもOK」というメッセージを表示。タイマー終了時は、作業を継続するか中断するかを選択できるUI（オープン・ループの維持を意識）にする。
